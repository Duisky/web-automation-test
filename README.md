# Web Automation Test Framework
基于 Python + Pytest + Selenium + Allure 的 Web UI 与 API 自动化测试框架。

## 技术栈
- Python 3.11
- Pytest（测试框架）
- Selenium（Web UI 自动化）
- Requests（接口自动化）
- Allure（测试报告）
- GitHub Actions（CI/CD）
- Docker（环境容器化）

## 项目结构
web-automation-test/
├── config/          # 环境配置
├── common/          # 公共封装
├── page_objects/    # 页面对象（POM）
├── api_tests/       # 接口测试用例
├── ui_tests/        # UI 测试用例
├── test_datas/      # 测试数据（YAML）
├── reports/         # Allure 测试报告
└── README.md        # 项目说明
